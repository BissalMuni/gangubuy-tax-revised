# 농지 외 부동산 유상거래 취득세율

> 농지·주택을 제외한 일반 부동산(상가, 토지, 사무실 등)을 유상거래로 취득하는 경우 적용되는 세율 안내

---

## 1. 적용 대상

1. **농지 및 주택을 제외한 부동산**을 **유상거래**(매매, 교환 등)로 취득하는 경우
2. 적용 대상 부동산 유형:
   - **상가** (근린생활시설, 판매시설 등)
   - **사무실** (업무시설)
   - **공장** (공장용 건물 및 부속토지)
   - **토지** (농지 외 일반토지, 임야 등)
   - **오피스텔** (사무용으로 사용하는 경우)
   - **무허가주택** (주택법상 주택이 아닌 것)
   - 기타 건축물 (창고, 주차장 등)

## 2. 주요 내용

### 가. 기본 세율

- 농지 외 부동산 유상거래 취득세 본세: **4%**
- 부동산 취득세의 **표준세율**에 해당
- 과세표준: **사실상 취득가격** (실거래가 기준)

### 나. 부가세 포함 합계세율

- 취득세 본세 **4%** + 농어촌특별세 **0.2%** + 지방교육세 **0.4%**
- 합계: **4.6%**

### 다. 무허가주택 적용

- 무허가주택(무허가건축물)은 주택법상 **주택으로 보지 않음**
- 따라서 주택 취득세율(1~3%)이 아닌 **일반 부동산 세율 4.6%** 적용
- 건축물대장에 등재되지 않은 건물도 사실상 취득 시 과세 대상

### 라. 오피스텔 적용

- **사무용 오피스텔**: 업무시설로 분류되어 **4.6%** 적용
- **주거용 오피스텔**: 실제 주거용으로 사용 시 주택 세율 적용 가능 (별도 판단 필요)
- 건축물대장상 용도와 실제 사용 용도가 다를 경우 **사실상 용도** 기준

## 3. 세율·금액

| 구분 | 세율 | 과세표준 | 비고 |
|------|------|----------|------|
| 취득세 본세 | **4%** | 사실상 취득가격 | 지방세법 §11①1가 |
| 농어촌특별세 | 0.2% | 취득세 과세표준 | |
| 지방교육세 | 0.4% | 취득세액의 20% 상당 | |
| **합계** | **4.6%** | - | 상가·사무실·공장·토지 등 |
| 무허가주택 | **4.6%** | 사실상 취득가격 | 주택법상 주택 아님 |
| 오피스텔(사무용) | **4.6%** | 사실상 취득가격 | 업무시설 분류 |

## 5. 주의사항

<Callout type="caution">

- **주택 vs 비주택 판단**이 세율에 큰 영향을 미치므로, 건축물의 실제 용도를 정확히 확인할 것
- 무허가주택은 주택이 아니므로 **다주택 중과세율 적용 대상에서 제외**되나, 반대로 주택 경감세율(1~3%)도 적용 불가
- 오피스텔의 경우 **건축물대장상 용도**와 **실제 사용 용도**가 다를 수 있으므로 사실관계 확인 필요
- 토지 취득 시 개발부담금, 농지전용부담금 등 **별도 부담금**이 발생할 수 있음
- 법인의 상가·사무실 취득 시에도 동일한 **4.6%** 적용 (주택과 달리 법인 중과 없음)

</Callout>
