# 농지 외 부동산 원시취득 세율

> 건물 신축 등 원시취득으로 부동산을 취득하는 경우 적용되는 세율 및 과세표준 안내

---

## 1. 적용 대상

1. **건물 신축**, 증축, 개축 등으로 부동산을 **원시취득**하는 경우
2. 원시취득의 유형:
   - 건물 **신축** (주거용·비주거용 모두 포함)
   - 건물 **증축** (기존 건물에 면적 추가)
   - **매립·간척**에 의한 토지 조성
   - 공유수면 매립 등
3. 다주택자라도 원시취득은 **별도 세율체계** 적용 (주택 중과세율 비적용)

## 2. 주요 내용

### 가. 기본 세율

- 원시취득 취득세 본세: **2.8%**
- 승계취득(유상 4%, 상속 2.8%, 증여 3.5%)과 구분되는 별도 세율

### 나. 부가세 포함 합계세율

- 취득세 본세 **2.8%** + 농어촌특별세 **0.2%** + 지방교육세 **0.16%**
- 합계: **3.16%**

### 다. 과세표준 산정

- 원칙: **사실상 취득가격** (실제 건축에 소요된 총비용)
- 개인 시공 시 취득가격 입증이 곤란한 경우: **시가표준액** 과세 가능
- 사실상 취득가격에 포함되는 비용 (PDF Sec 26, p52-53):
  1. **도급계약서** 상 공사비
  2. **설계비**
  3. **감리비**
  4. **시설물설치비**
  5. **전기·수도·가스 인입공사비**
  6. **등록면허세**
  7. **철거비** (기존 건물 철거 시)
  8. **기반시설부담금**

### 라. 다주택자 원시취득 특례

- 다주택자가 주택을 신축하는 경우에도 원시취득 세율 **2.8%** 적용
- 주택 유상거래 중과세율(8~12%)과 무관하게 **원시취득 별도 세율체계** 적용
- 다만, 신축 후 취득세 신고 시 주택 수 산정에는 포함될 수 있음

## 3. 세율·금액

| 구분 | 세율 | 과세표준 | 비고 |
|------|------|----------|------|
| 취득세 본세 | **2.8%** | 사실상 취득가격 | 지방세법 §11①3 |
| 농어촌특별세 | 0.2% | 취득세 과세표준 | |
| 지방교육세 | 0.16% | 취득세액의 20% 상당 | |
| **합계** | **3.16%** | - | - |
| 개인시공(입증곤란) | **3.16%** | 시가표준액 | 시가표준액 과세 |

## 5. 주의사항

<Callout type="caution">

- 신축 건물의 과세표준은 **실제 건축비용 전체**이며, 도급계약서 금액뿐만 아니라 설계비, 감리비, 인입공사비 등 **부대비용 모두 포함**
- 개인이 직접 시공하여 취득가격 입증이 곤란한 경우 **시가표준액**으로 과세될 수 있으므로, 건축 관련 증빙서류를 철저히 보관할 것
- 다주택자라도 원시취득 시 중과세율이 아닌 **2.8% 적용**되나, 이후 해당 주택 매각·보유 시 다주택 관련 규정은 별도 적용
- 증축의 경우 증축 부분만 원시취득에 해당하며, 기존 건물 부분은 해당 없음
- 사용승인일(임시사용승인일) 또는 사실상 사용일 중 빠른 날이 **취득시기**임

</Callout>
