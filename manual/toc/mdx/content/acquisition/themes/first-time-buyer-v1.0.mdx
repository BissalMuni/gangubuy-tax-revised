# 2. 생애최초 주택취득 감면

> 생애 처음 주택을 취득하는 경우 취득세를 100% 면제 (한도 200~300만원)하는 제도. 2020.8.12 신설.

---

## 1. 감면 대상 및 한도

| 대상 주택 | 조건 | 감면 |
|----------|------|------|
| 전용 60m² 이하 공동주택 (아파트 제외), 도시형생활주택 | 취득가액 6억원(수도권) 이하 | 취득세 **100% 면제** (한도 **300만원**) |
| 다가구주택 | 건축물대장 호수별 전용면적 구분 기재, 전용 60m² 이하 호수 한정, 6억원(수도권) 이하 | 취득세 **100% 면제** (한도 **300만원**) |
| 상기 외 주택 | 취득가액 **12억원** 이하 | 취득세 **100% 면제** (한도 **200만원**) |

<Callout type="info">

- 혼인여부, 연령과 **무관** (단 미성년 만19세 미만 제외)
- 소득제한 **없음**, 면적제한 **없음**
- **유상거래**(경매 포함)만 인정, 부담부증여는 **제외**
- 외국인 감면 **안 됨**
- 농특세: 상가나 85m² 이상 주택은 감면세액의 20%

</Callout>

---

## 2. 감면 요건

1. 주택 취득일 현재 **본인 및 배우자**가 주택을 소유한 사실이 없는 경우
   - 혼인이 확인되는 외국인 배우자 포함
   - 취득자가 만 19세 미만 → 감면 **제외**
   - 취득자의 배우자가 취득일 현재 주택을 소유하였거나 처분한 경우 → 감면 **제외**
2. 취득당시 가액이 **12억원 이하** 주택 유상거래 (부담부증여 제외)
3. 주택취득자가 **실제 거주**해야 함

### 주택의 범위

- 단독주택, 아파트, 다세대, 연립 등 공동주택 (주택법 §2①)
- **제외**: 오피스텔, 분양권
- 무허가주택도 소유한 것으로 **보지 않음**

---

## 3. 추징요건

| 번호 | 추징 사유 |
|------|----------|
| ① | 주택 취득일부터 **3개월 이내** 실거주를 시작하지 아니하는 경우 (전입신고 필요) |
| ② | 주택 취득일부터 **3개월 이내** 1가구 1주택이 되지 아니하는 경우 |
| ③ | 실거주 기간 **3년 미만** 상태에서 해당 주택을 매각·증여, 다른 용도(임대)로 사용하는 경우 |

<Callout type="caution">

- 전·월세, 갭투자 등 실거주 목적이 아닌 주택 구입은 혜택 **배제**
- 중과규정(지방세법 §13조의2 세율) **적용하지 않음**

</Callout>

---

## 4. 예외조항 — 주택 소유 사실이 없는 것으로 보는 경우 (§36의3③)

| 번호 | 예외 사유 |
|------|----------|
| 1 | **상속**으로 공유지분을 소유하였다가 모두 처분한 경우 |
| 2 | 도시지역 외 지역에 소재한 주택소유자가 타 지역으로 이주한 경우 (취득일부터 3개월내 처분 조건) |
| | - 20년 이상 경과된 단독주택, 85m² 이하 단독주택, 상속주택 |
| 3 | 전용면적 **20m² 이하** 주택을 소유하고 있거나 처분한 경우 (2개 이상 소유 시 제외) |
| 4 | 취득일 현재 시가표준액 **100만원 이하** 주택을 소유하고 있거나 처분한 경우 |
| 5 | **전세사기피해주택**을 소유하고 있거나 처분한 경우 |
| 6 | 취득당시가액 3억원(수도권) 이하, 임차인으로서 1년 이상 거주한 주택을 2024.1.1~2025.12.31 기간 중 취득하여 감면받은 경우 |
