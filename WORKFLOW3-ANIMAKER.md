# Animaker 애니메이션 제작 워크플로우

## 개요
생성된 시나리오를 바탕으로 Animaker에서 쇼츠 영상을 제작하는 워크플로우

## 입력
```
scenarios/[문서명]/[대분류]/[중분류]/[세부항목]/
├── scenario.json       # 장면별 시나리오
├── script.md           # 나레이션 스크립트
└── storyboard.md       # 스토리보드
```

## 출력
```
exports/[문서명]/[세부항목]/
├── [세부항목]_final.mp4    # 최종 영상
├── [세부항목]_thumbnail.png # 썸네일
└── metadata.json           # 영상 메타데이터
```

## Animaker 프로젝트 설정

### 영상 규격 (쇼츠용)
- **해상도**: 1080 x 1920 (9:16 세로)
- **길이**: 60초 이하
- **프레임레이트**: 30fps

### 프로젝트 템플릿 구성
```
templates/
├── intro_template/      # 인트로 장면 템플릿
├── comparison_template/ # 비교 장면 템플릿
├── example_template/    # 예시 장면 템플릿
└── outro_template/      # 아웃트로 장면 템플릿
```

## 제작 단계

### 1단계: Animaker 프로젝트 생성
1. Animaker 접속 (https://www.animaker.com)
2. "Create" → "Vertical Video" 선택
3. 프로젝트명: `[문서명]_[세부항목]`

### 2단계: 장면별 제작

#### 장면 1: 인트로 (5초)
- 배경: 세금/법률 관련 배경
- 요소:
  - 타이틀 텍스트 (애니메이션 효과)
  - 연도 표시 "2026"
  - 아이콘 (세금, 법률 관련)
- 음악: 경쾌한 인트로 사운드

#### 장면 2: 변경 전 (15초)
- 배경: 흐린/과거 느낌
- 요소:
  - "변경 전" 라벨
  - 캐릭터 (고민하는 표정)
  - 말풍선 또는 텍스트 박스
- 색상 톤: 회색/파란색 계열

#### 장면 3: 변경 후 (20초)
- 배경: 밝은/현대적 느낌
- 요소:
  - "변경 후" 라벨
  - 캐릭터 (기쁜 표정)
  - 강조 애니메이션
- 색상 톤: 녹색/노란색 계열

#### 장면 4: 예시 (15초)
- 배경: 중립적
- 요소:
  - 구체적 숫자/금액 표시
  - 계산 애니메이션
  - 아이콘 (돈, 집, 차 등)

#### 장면 5: 아웃트로 (5초)
- 배경: 브랜드 컬러
- 요소:
  - 요약 텍스트
  - 구독/좋아요 아이콘
  - 채널 로고

### 3단계: 나레이션 추가
1. script.md 텍스트 복사
2. Animaker 보이스오버 기능 사용
   - TTS 사용 또는 직접 녹음
3. 장면과 동기화

### 4단계: 음악/효과음 추가
- 배경음악: 저작권 프리 음악
- 효과음: 전환, 강조 효과

### 5단계: 검토 및 내보내기
1. 미리보기로 전체 확인
2. 타이밍 조정
3. MP4로 내보내기 (1080p)

## 에셋 라이브러리

### 추천 캐릭터
- 비즈니스맨/비즈니스우먼
- 공무원 스타일
- 일반 시민

### 추천 배경
- 사무실
- 법원/관공서
- 집/부동산

### 추천 아이콘
- 돈/화폐
- 세금 관련 (영수증, 계산기)
- 법률 관련 (저울, 망치)
- 부동산 (집, 건물, 토지)

## 업로드 가이드

### YouTube Shorts
- 제목: `[세금 종류] 2026년 이렇게 바뀝니다 #개정세법 #지방세`
- 설명: 개정 내용 요약 + 채널 소개
- 태그: 지방세, 개정세법, 2026, 세금, 취득세 등

### 썸네일 제작
- 크기: 1080 x 1920
- 요소: 핵심 키워드 + 아이콘 + 강조 색상
- 텍스트: 크고 읽기 쉽게
